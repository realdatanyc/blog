## Create your own blog on Github
The main advantage for creating your own blog on Github is that it doesn't cost you a monthly fee.  Here's where I learned how: [Building a Free Blog with GitHub Pages in Minutes](https://chadbaldwin.net/2021/03/14/how-to-build-a-sql-blog.html)

Another advantage is that it doesn't contain ads.  Also, you can easily update the pages, edit, and add more things.
